## Общие коды
```
10-0 - Конец связи
10-1 - Не понял/повторите
10-2 - Отказ / Невозможно выполнить
10-3 - Прекратить вещание на частоте
10-4 - Принял / Так точно
10-5 - Важное сообщение / Внимание
10-6 - Перекличка / Откликнуться на частоте

10-7 - Покинул службу без уточнения причины 
 -А  - По причине: сон/отдых/питание
 -Б  - По причине: лечение/починка/пополнение боезапаса
 -В  - По причине: спец. приказ/личная причина

10-8 - Заступил на службу

10-9 - Отстранён от службы без уточнения причины 
 -А  - По причине: сон/отдых/питание
 -Б  - По причине: лечение/починка/пополнение боезапаса
 -В  - По причине: спец. приказ/личная причина

10-10- Требуется эвакуация
 -А  - Раненных
 -Б  - Гражданских / Заложников / Пленных (вражеские лица)
 -В  - Снаряжения/ресурсов

10-11- Контакт/Обнаружение
 -Г - Гражданский 
 -П - Противник 
 -Т - Техника
 -С - Снаряжение 
 -З - Заражение

10-15- Сектор чист
10-16- Наши силы обнаружены противником
10-17- Запрашиваю инструкции / Жду приказов
10-18- Возвращаюсь на базу
10-19- Уточните маршрут к позиции (указанной в 10-20)
10-20- Указать местоположение
```

## Грузы
```
100 - Боеприпасы
200 - Убитые / Погибшие
300 - Раненые
400 - Контуженные / Военнопленные
500 - Медикаменты
600 - Ценные грузы (документы, артефакты, образцы и т.п.)
700 - Гражданский груз (Ресурсы, еда, т.п.)
800 - Оружие массового поражения
900 - Опасный / нестабильный груз
```
